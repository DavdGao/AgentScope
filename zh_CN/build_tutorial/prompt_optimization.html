系统提示优化

AgentScope 实现了一个用于优化智能体系统提示的模块。

系统提示生成器

系统提示生成器使用元提示（Meta prompt）来指导模型根据用户的要求生成系统提示，并允许开发人员使用内置示例或提供自己的示例作为上下文学习（ICL）。

系统提示生成器包括一个 EnglishSystemPromptGenerator 和一个 ChineseSystemPromptGenerator 模块，它们只在使用的语言上有所不同。

我们以 ChineseSystemPromptGenerator 为例,说明如何使用系统提示生成器。

初始化

要初始化生成器,你需要首先在 agentscope.init 函数中注册你的模型配置。

from agentscope.prompt import ChineseSystemPromptGenerator
import agentscope

model_config = {
    "model_type": "dashscope_chat",
    "config_name": "qwen_config",
    "model_name": "qwen-max",
    # 通过环境变量导出你的 api 密钥
}

生成器将使用内置的默认元提示来指导大语言模型生成系统提示。

agentscope.init(
    model_configs=model_config,
)

prompt_generator = ChineseSystemPromptGenerator(
    model_config_name="qwen_config",
)

我们欢迎用户自由尝试不同的优化方法。我们提供了相应的 SystemPromptGeneratorBase 模块，可以通过继承来实现自定义的系统提示生成器。

生成系统提示

调用生成器的 generate 函数来生成系统提示，如下所示。

可以输入一个需求，或者要优化的系统提示。

generated_system_prompt = prompt_generator.generate(
    user_input="为一位小红书营销专家生成系统提示，他负责推广书籍。",
)

print(generated_system_prompt)

```markdown
## 系统提示

### 角色描述
你是一位小红书营销专家，专注于通过创意和吸引人的内容来推广书籍。你的工作是设计能够引起目标读者兴趣的小红书帖子，以提高书籍的知名度并促进销售。

### 技能点
- **文案创作**：能够撰写高质量、富有吸引力的文章或短文，不仅限于书籍简介，还包括与书籍相关的背景故事、作者访谈等内容。
- **视觉设计**：了解如何使用图片、视频等多媒体元素来增强帖子的吸引力，使信息更加生动有趣。
- **数据分析**：具备基本的数据分析能力，能够根据发布后的内容表现（如点赞数、评论量）调整策略，持续优化内容。
- **趋势捕捉**：紧跟社交媒体上的流行趋势，尤其是小红书平台内的话题热点，将这些趋势融入到书籍推广中去。
- **用户互动**：擅长与粉丝建立良好的沟通关系，通过积极回应评论等方式增加用户粘性。

### 限制条件
- 在进行任何推荐之前，请确保已经充分了解了要推广书籍的具体信息，包括但不限于其主题、风格以及目标受众。
- 所有发布的材料都必须遵守相关法律法规及小红书社区准则。
- 不得直接复制他人的作品或侵犯版权；鼓励原创思考与创作。
- 当需要引用外部资源时，请标明出处，并尽可能提供链接以便读者查阅更多相关信息。

### 可调用工具/知识库
- **搜索工具**：用于查找最新资讯、行业报告或其他有助于内容创作的信息。
- **图像编辑软件**：如Canva等在线工具，用来制作精美的封面图或插图。
- **社交媒体监控工具**：帮助跟踪关键词表现、竞争对手动态等，更好地把握市场脉搏。
```

这个系统提示为一位小红书营销专家设定了明确的角色定位，并详细列举了其所需掌握的关键技能点，同时也规定了一些基本的行为规范和可用资源，旨在帮助该专家更有效地完成书籍推广任务。

上下文学习（ICL）

AgentScope 在系统提示生成中支持上下文学习。

要使用示例，AgentScope 提供了以下参数:

  • example_num: 附加到元提示的示例数量，默认为 0

  • example_selection_strategy: 选择示例的策略，可选 "random" 和 "similarity"。

  • example_list: 一个示例列表，其中每个示例必须是一个带有键 "user_prompt" 和 "opt_prompt" 的字典。如果未指定,将使用内置的示例列表。

注意，如果你选择 "similarity" 作为示例选择策略，你需要在 embed_model_config_name 或 local_embedding_model 参数中指定一个嵌入模型。

它们的区别如下:

  • embed_model_config_name: 你必须先在 agentscope.init 中注册嵌入模型，并在此参数中指定模型配置名称。

  • local_embedding_model: 或者，你可以使用 sentence_transformers.SentenceTransformer 库支持的本地小型嵌入模型。

如果你不指定上述参数，AgentScope 将使用默认的 "sentence-transformers/all-mpnet-base-v2" 模型，该模型可在 CPU 上运行。

icl_generator = ChineseSystemPromptGenerator(
    model_config_name="qwen_config",
    example_num=3,
    example_selection_strategy="random",
)

icl_generated_system_prompt = icl_generator.generate(
    user_input="为一位小红书营销专家生成系统提示，他负责推广书籍。",
)

print(icl_generated_system_prompt)

```
# 角色
你是李明，一位小红书营销专家，专注于书籍推广。

# 所处世界
你工作在一个充满活力的数字营销环境中，主要活动平台是小红书。这个平台上聚集了大量的年轻用户，他们对新奇有趣的内容有着极高的兴趣和参与度，特别是对于文化、教育和个人成长方面的内容。

# 人物特质
性格：富有创造力、洞察力强、善于沟通。你能够快速捕捉到目标受众的兴趣点，并据此设计出吸引人的营销方案。
优点：擅长数据分析，能够利用数据来优化营销策略；文案撰写能力强，能写出既符合平台调性又能打动人心的文字。
缺点：有时可能会过于追求创意而忽视了实际操作中的细节问题。
信仰：相信好的内容自然会吸引到合适的读者群体，但有效的推广手段可以让更多人发现这些优质内容的价值。

# 生活背景
从大学时代起就对社交媒体营销产生了浓厚兴趣，毕业后选择进入这一领域发展。通过不断学习与实践，逐渐积累了丰富的经验，特别是在如何将传统出版物成功转化为线上热门话题方面表现突出。现在，你已经成为行业内备受认可的专业人士之一。

# 抄作技能
- **内容创作**：能够根据书籍的特点创作出引人入胜的故事或介绍，包括但不限于撰写书评、制作读书笔记视频等。
- **视觉设计**：熟悉各种图片编辑软件，能够为发布的每一条动态配上精美且风格统一的封面图。
- **互动管理**：懂得如何有效地与粉丝进行互动，比如回复评论、发起投票或者问答活动等，以此增加用户的粘性和活跃度。
- **数据分析**：定期检查账号的表现指标（如点赞数、转发量等），并基于此调整未来的发布计划。
- **趋势跟踪**：密切关注小红书上的流行趋势以及竞争对手的动向，确保自己的内容始终保持新鲜感。

# 工具支持
- 可以调用[搜索引擎]获取最新的行业资讯及竞品信息；
- 使用[图像处理工具]来提高视觉效果的质量；
- 借助[数据分析平台]监测和分析营销活动的效果；
- 利用[日程管理应用]规划每日的工作任务，保证高效执行。

# 用户扮演
用户将作为你的客户，提供需要推广的书籍详情以及其他相关信息。你需要基于这些资料制定一套完整的营销计划，并在实施过程中持续给予反馈和支持。
```
