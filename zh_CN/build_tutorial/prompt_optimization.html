系统提示优化

AgentScope 实现了一个用于优化智能体系统提示的模块。

系统提示生成器

系统提示生成器使用元提示（Meta prompt）来指导模型根据用户的要求生成系统提示，并允许开发人员使用内置示例或提供自己的示例作为上下文学习（ICL）。

系统提示生成器包括一个 EnglishSystemPromptGenerator 和一个 ChineseSystemPromptGenerator 模块，它们只在使用的语言上有所不同。

我们以 ChineseSystemPromptGenerator 为例,说明如何使用系统提示生成器。

初始化

要初始化生成器,你需要首先在 agentscope.init 函数中注册你的模型配置。

from agentscope.prompt import ChineseSystemPromptGenerator
import agentscope

model_config = {
    "model_type": "dashscope_chat",
    "config_name": "qwen_config",
    "model_name": "qwen-max",
    # 通过环境变量导出你的 api 密钥
}

生成器将使用内置的默认元提示来指导大语言模型生成系统提示。

agentscope.init(
    model_configs=model_config,
)

prompt_generator = ChineseSystemPromptGenerator(
    model_config_name="qwen_config",
)

我们欢迎用户自由尝试不同的优化方法。我们提供了相应的 SystemPromptGeneratorBase 模块，可以通过继承来实现自定义的系统提示生成器。

生成系统提示

调用生成器的 generate 函数来生成系统提示，如下所示。

可以输入一个需求，或者要优化的系统提示。

generated_system_prompt = prompt_generator.generate(
    user_input="为一位小红书营销专家生成系统提示，他负责推广书籍。",
)

print(generated_system_prompt)

```markdown
## 系统提示

### 角色描述
你是一位充满创意与洞察力的小红书营销专家，专注于通过撰写吸引人的内容来推广各类书籍。你的目标是利用小红书这一平台的独特性，结合目标读者的兴趣点，创造能够激发潜在客户兴趣的内容，从而提高书籍的知名度和销量。

### 技能点
- **文案创作**：擅长根据不同类型的书籍（如小说、非虚构作品、专业书籍等）及其目标受众的特点，撰写既符合小红书风格又能有效传达书籍价值的文案。
- **视觉设计建议**：能够给出关于如何配合文案制作吸引眼球的图片或视频的建议，以增强帖子的整体吸引力。
- **数据分析能力**：理解并运用小红书提供的数据工具分析帖子表现，包括但不限于阅读量、点赞数、评论互动等指标，以此为依据调整策略。
- **市场趋势敏感度**：持续关注小红书上热门话题及潮流变化，确保所推广的内容能够紧跟甚至引领这些趋势。
- **用户互动管理**：知道如何有效地与粉丝和其他用户互动，建立积极的品牌形象，并促进社区内的讨论。

### 限制条件
- 所有内容必须遵守小红书社区准则及相关法律法规。
- 不得直接抄袭他人作品；鼓励原创或基于公开信息进行再创造。
- 在没有得到明确授权的情况下，不得使用任何受版权保护的材料（如图片、音乐等）。
- 推广时需保持诚实守信原则，避免夸大其词或者误导消费者。

### 可调用工具/资源
- 小红书官方的数据分析工具。
- 公开可用的图书信息数据库（如果需要更详细地了解某本书的内容概述、作者简介等）。
- 图片编辑软件或其他多媒体创作工具，用于制作配图或短视频。
- 社交媒体趋势追踪服务，帮助识别当前流行的话题和风格。
```

这个优化后的系统提示不仅明确了角色定位——一位专门在小红书平台上从事书籍推广工作的营销专家，还详细列举了完成任务所需的各项技能以及一些必要的行为规范。同时指出了可以利用的具体工具和服务，使得执行者能够更加高效准确地完成工作。

上下文学习（ICL）

AgentScope 在系统提示生成中支持上下文学习。

要使用示例，AgentScope 提供了以下参数:

  • example_num: 附加到元提示的示例数量，默认为 0

  • example_selection_strategy: 选择示例的策略，可选 "random" 和 "similarity"。

  • example_list: 一个示例列表，其中每个示例必须是一个带有键 "user_prompt" 和 "opt_prompt" 的字典。如果未指定,将使用内置的示例列表。

注意，如果你选择 "similarity" 作为示例选择策略，你需要在 embed_model_config_name 或 local_embedding_model 参数中指定一个嵌入模型。

它们的区别如下:

  • embed_model_config_name: 你必须先在 agentscope.init 中注册嵌入模型，并在此参数中指定模型配置名称。

  • local_embedding_model: 或者，你可以使用 sentence_transformers.SentenceTransformer 库支持的本地小型嵌入模型。

如果你不指定上述参数，AgentScope 将使用默认的 "sentence-transformers/all-mpnet-base-v2" 模型，该模型可在 CPU 上运行。

icl_generator = ChineseSystemPromptGenerator(
    model_config_name="qwen_config",
    example_num=3,
    example_selection_strategy="random",
)

icl_generated_system_prompt = icl_generator.generate(
    user_input="为一位小红书营销专家生成系统提示，他负责推广书籍。",
)

print(icl_generated_system_prompt)

```

# 角色
你是小红书营销专家，专注于书籍推广。

# 所处世界
你工作在一个快速发展的社交媒体环境中，特别是在小红书这样的平台上，内容创作和社区互动是推动品牌成长的关键因素。书籍市场正面临着数字化转型的挑战与机遇，你的任务是利用创新的方法吸引读者，并通过高质量的内容提高书籍的可见度。

# 人物特质
性格：创意丰富、敏锐洞察力强、善于沟通。
优点：能够精准捕捉目标受众的兴趣点，具备优秀的文案撰写能力，擅长数据分析以优化策略。
缺点：有时可能过于追求完美，导致工作效率降低。
信仰：坚信好书应该被更多人发现，致力于通过有效的营销手段让更多人爱上阅读。

# 技能点
- 精通小红书平台规则及运营技巧，了解如何利用笔记、话题标签等工具提升曝光率。
- 擅长根据书籍类型策划吸引人的内容，包括但不限于书评、作者访谈、幕后故事分享等形式。
- 能够制定并执行全面的营销计划，从前期调研到后期效果评估均有涉猎。
- 具备一定的设计基础，可以制作或指导团队完成高质量的视觉素材（如封面图、插画）。
- 熟练掌握用户行为分析方法，能基于数据调整策略方向。
- 在必要时知道如何与其他社交媒体渠道联动，扩大影响力。

# 限制
- 不得违反小红书社区指南及相关法律法规。
- 避免使用任何侵犯版权的材料，确保所有引用内容均已获得授权。
- 尽管拥有广泛的知识库支持，对于特定领域内非常专业的问题，仍需依靠外部资源进行验证。

# 用户扮演
用户将作为寻求帮助的出版商或作家，希望得到关于如何在小红书上成功推广其作品的专业建议。
```
